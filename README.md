# AORTA AI - Predicting AAA Post-Operative Complications

This project is a Machine Learning prototype designed to assist surgeons in predicting the risk of complications following Abdominal Aortic Aneurysm (AAA) surgery.

## Prototype Objectives
- Analyze patient data (age, BMI, aneurysm diameter, etc.).
- Predict the probability of post-operative complications.
- Provide a simple interface for non-technical medical staff.

## Installation
1. Clone the repository.
2. Install the dependencies: `pip install -r requirements.txt`
3. Run the script: `python app.py`
